Usage Guide

1. Initialize the component

// Create TagList instance with mount element ID
const tagList = new TagList('mountTagList');

2. Add tags using add_item()

// Add a filter tag
tagList.add_item({"id": 0, "type": "filter", "label": "closest"});

// Add a generic tag
tagList.add_item({"id": 21, "type": "tag", "label": "fade"});

// Add a barber tag
tagList.add_item({"id": 23, "type": "barber", "label": "alex_barber"});

// Add a barbershop tag
tagList.add_item({"id": 12, "type": "barbershop", "label": "Portsmouth Fade House"});

3. Get all current tags using get_items()

// Returns JSON array
const currentTags = tagList.get_items();
console.log(currentTags);

// Example output:
// [
//   {"id": 0, "type": "filter", "label": "closest"},
//   {"id": 21, "type": "tag", "label": "fade"},
//   {"id": 23, "type": "barber", "label": "alex_barber"}
// ]
