app/static/css/pages/signin.css
===============================

Overview
--------

app/static/css/pages/signin.css controls layout, spacing, typography, colors, and interaction states for its target UI surface. It includes 4 media query block(s) to adapt the interface across viewport sizes. The file is loaded by page templates alongside app/static/css/main.css and other component/page styles as needed.

Purpose
-------

This stylesheet defines presentation rules for the signin page. It styles selectors such as .login-page, .login-card, .login-card__header, and .login-card__title.
