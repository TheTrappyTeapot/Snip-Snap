app/db.py
=========

Overview
--------

It opens PostgreSQL connections and implements query helpers used by route handlers and JSON APIs throughout the app. The module includes profile updates, gallery/photo writes, discover feed retrieval, map/search data queries, and review persistence. Most web endpoints delegate persistence work to these functions to keep request handlers thin.

Purpose
-------

This module in `app/db.py` provides backend application behavior. Function responsibilities: `_get_conn` create and return a PostgreSQL connection from DATABASE_URL; `create_app_user` create a new App_User record and return the user_id; `link_auth_user_id` link Supabase auth user UUID to App_User row (only if not already linked); `get_app_user_by_auth_user_id` retrieves Flask app instance user by auth user ID; `get_app_user_by_email` retrieves Flask app instance user by email; `get_user_promo` get user profile data for the userPromo component; `get_barber_public_by_user_id` public barber profile (safe fields only); `get_barber_id_from_user_id` get the barber_id for a given user_id; `get_barbershop_by_id` get barbershop details including all barbers working there; `get_shifts_for_barber` get all shifts for a barber, ordered by day of week; `add_shift` add a shift for a barber; `delete_shift` delete a shift, only if it belongs to the given barber; `get_shop_opening_hours` get aggregate opening hours for a shop by analyzing all barbers' shifts; `update_barber_profile` barber edits their own public profile; `update_barber_bio` update the bio for a barber; `update_barbershop_website` update the website for the barbershop the barber works at; `update_barber_social_links` update the social_links JSON for a barber; `filter_existing_tag_ids` filters existing tag ids; `create_haircut_post` creates haircut post; `fetch_discover_posts` fetches discover post data; `update_user_location` updates user location; `get_user_location` retrieves user location; `update_user_postcode` updates user postcode; `get_user_postcode` retrieves user postcode; `_pick_label` chooses label; `get_barbershops_for_map` return all barbershops with their barbers for the map page API; `fetch_discover_search_items` fetches discover search items list; `get_all_barbershops` get all barbershops for autocomplete in profile; `get_barber_barbershop` get the barbershop that a barber works at; `update_user_profile` update user profile (username, postcode, role, and optionally latitude/longitude); `create_or_update_barber` create a barber record if it doesn't exist, or update the barbershop if it does; `update_barber_barbershop` update the barbershop that a barber works at; `get_reviews_for_barber` fetch all reviews for a specific barber from the database; `submit_barber_review` insert a new review into the Reviews table; `postcode_to_coordinates` convert UK postcode to latitude/longitude using postcodes.io API; `create_barbershop` create a new barbershop in the database; `update_or_create_profile_photo` create or update a ProfilePhoto record for a user; `get_profile_photo` fetch the ProfilePhoto record for a user; `get_barber_gallery_photos` fetch haircut photos for a barber that are NOT posts (is_post = false); `get_barbershop_gallery_photos` fetch haircut photos for all barbers at a barbershop that are NOT posts (is_post = false).
